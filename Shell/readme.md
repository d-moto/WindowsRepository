## スクリプト名
extract_log.sh - syslogファイルから指定された期間のログを抽出するシェルスクリプト

## 使用法
```
$ ./extract_log.sh [start_time] [end_time]
```

- [start_time] : 検索開始時刻を表す引数。フォーマットは "MM,DD,hh,mm" (例: "04,08,09,02")
- [end_time] : 検索終了時刻を表す引数。フォーマットは "MM,DD,hh,mm" (例: "04,08,09,30")

## 出力
抽出されたログが保存されたファイルが以下の場所に作成されます。
```
./log/syslog.[YYYYMMDDHHMMSS].log
```
- YYYYMMDDHHMMSS : 抽出処理を実行した時刻を表すフォーマットです。

## 動作
1. 引数の数が2つでない場合、使用法を表示してスクリプトを終了します。
2. 引数のフォーマットをチェックします。不正なフォーマットの場合、エラーメッセージを表示してスクリプトを終了します。
3. syslogファイルを検索し、指定された期間のログを抽出します。
4. 抽出されたログを、ログファイルとして保存します。

## エラーメッセージ
- "Usage: extract_log.sh [start_time] [end_time]" : 引数の数が正しくありません。
- "Error: Argument 'XX' is not a number" : 引数に数字以外が含まれています。
- "Error: Argument 'XX' has too many digits" : 引数の桁数が不正です。
- "Error: Argument 'XX' is out of range (0-59)" : 引数の範囲が不正です。

## 注意事項
- 引数のフォーマットが不正な場合、スクリプトは強制終了します。
- 検索期間に該当するログが存在しない場合、ログファイルは作成されません。
- 既に同じファイル名が存在する場合、ファイルは上書きされます。
